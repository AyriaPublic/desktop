# Ayria plugin

## Package
All the files for an Ayria plugin are packed in a ZIP archive file with no compression. Inside it: binary file(s) and a plugin manifest file.

## Manifest
An example manifest:
```yaml
name: Kicking ninja's
version: 5.0.3
description: ''
games:
- platform: steam
  id: 219740
- platform: steam
  id: 431730
```
